Stable Diffusion 绘图

发表于2023-05-01|更新于2023-05-02|Tutorial
|字数总计:2,176|阅读时长:9分钟

环境配置

暂时没有带有GPU的Linux操作系统，本文都在window11下实验的

运行环境：python==3.10.9 torch==1.13.1+cu117 torchvision==0.14.1+cu117，所以先要安装python-3.10.9、cuda-11.7、torch-1.13.1、torchvision-0.14.1

安装 Python

按照操作系统下载对应的 Python-3.10.9，后安装，安装时记得勾选Add Python 3.10.9 to PATH

安装Python时Pip会默认安装

安装 CUDA

cuda 的安装需要与你电脑的显卡驱动版本相对应，这里需要的 cuda 版本 为 11.7 及以上的，

查看显卡支持的 cuda 版本：

点击NVIDIA控制面板->帮助->系统信息，如果 cuda 版本不够，建议升级一下显卡驱动

CUDA下载链接： CUDA Toolkit Archive， 找到合适自己的版本下载安装

安装 torch

torch官网下载：PyTorch，选择与 cuda 对应的版本。选择利用Conda还是Pip安装都行

如果使用Conda和Pip因为网络的问题安装不上torch，可以直接下载 .whl 文件进行安装，下载地址 cu117，需要注意的是

要对应上你的电脑系统、python版本，需要python3.10.9及以上

找到你下载的 .whl 文件，在这个文件夹中使用 cmd 窗口，输入命令：pip install 名称.whl 进行安装

下载 Stable Diffusion web UI

  1. 下载：Stable Diffusion web UI 或者

使用git命令：git clone https://github.com/AUTOMATIC1111/stable-diffusion-webui.git

  1. 安装依赖：在Stable Diffusion web UI目录下的cmd窗口中

执行命令python -m pip install -r requirements.txt

执行上面的命令报错误的话

可以在cmd 中执行下面命令

再执行 python -m pip install -r requirements.txt

  1. 配置 stable-diffusion-webui

a. 编辑根目录下launch.py文件（可选）
将https://github.com替换为https://ghproxy.com/https://github.com，即使用Ghproxy代理，加速国内Git。

b. 都安装完成后运行webui.bat

根目录下将生成tmp和venv目录。

c. 编辑venv目录下pyvenv.cfg文件
将include-system-site-packages = false改为include-system-site-packages = true

d. 安装xformer（可选）

e. 安装语言包（可选）
将文件chinese-all-0408.json 和 chinese-english-0408.json放到目录/localizations目录中。
运行webui后进行配置，操作方法见链接：stable-diffusion-webui 的汉化版本。

f. 安装扩展（可选）
将sd-webui-controlnet解压缩到/extensions目录中。
将control_sd15_openpose.pth文件复制到/extensions/sd-webui-controlnet/models目录中。
不同的扩展可能还需要安装对应的系统，比如controlnet要正常使用则还需要安装ffmpeg等。

g. 安装模型（稍后）
下载的各种模型放在/models/Stable-diffusion目录中即可。

h. 再次执行根目录下webui.bat文件
用浏览器打开webui.bat所提供的网址即可运行。
其中提供了网址：http://127.0.0.1:7860
打开该网址后在Settings -> User interface -> Localization (requires restart)设置语言，在菜单中选择chinese-all--0408（前提是已经在目录中放入了对应语言包，见上），点击Apply Settings确定，并且点击Reload UI重启界面后即可。

注意：模型很大，至少需要4G以上的显存，没有这么大显存的（比如我）可以使用python launch.py --lowvram --always-batch-cond-uncond --opt-split-attention 代替 执行 webui.bat。这里需要在 NVIDIA控制面板->管理3D设置 将图形处理器设置为 集成图形

使用

下载 ChilloutMix 模型

ChilloutMix模型下载链接，需要将 ChilloutMix 模型放在 /stable-diffusion-webui/models/Stable-diffusion/ 文件夹下。

下载几个画像模型

  • korean doll likeness
  • taiwan doll likeness
  • japanese doll likeness
  • Lucy
  • EVA-凌波丽
  • eva asuka

下载后需要将这些画像模型放在/stable-diffusion-webui/models/Lora/ 文件夹下。

安装 kohya-ss Additional Networks 的外挂

  1. 在浏览器打开http://127.0.0.1:7860，然后选择chilloutmix_NiPrunedFp32Fix.safetensors，再点击 Extensions->Available->Load from，往下找到 kohya-ss Additional Networks，最后点击 install

  2. 安装kohya-ss Additional Networks 外挂完成后，点击Apply and restart UI就可以看的 Additional Networks的标签了

  3. 在txt2img页面的 Generate下面的第三个紫色按钮，点击它；再点击Lora可以看到我们放入的画像模型

AI网红照片的生成

  1. 点击Lora下面的korean doll likeness、taiwan doll likeness、japanese doll likeness画像模型，就会在提示词栏生成提示词，所有的lora画像模型的权重和不能超过1，注意调整。

  2. 接着需要输入提示词（你想得到的部分）和负面提示词（你不想看到的部分）

提示词示例：

负面提示词示例：

这些提示词可以通过这个网站查找civitai.com，点击这个网站上的图片就可以看到提示词了。

  1. 把取样模式调整为DPM++SDE Karras，Sampling steps设置为30，按下紫色按钮收起Additional Networks界面，点击Generate开始抽奖。

  1. 挑选自己想要的画像，按下绿色按钮就固定了训练的随机种子，再次点击Generate会以当前这张图画为基础生成其他画像，骰子按钮是不固定随机种子，每次产生的画像不一样。

  1. 点击 Send to img2img，将会以图生图，可以调整生成图形的大小（width，height），以及每次生成的图片数目（Batch count），挑选自己想要的画像，然后点击 Send to inpaint 对图片进行局部绘制。

  1. 在Send to inpaint中使用笔刷涂改想要修改的部分，然后点击Generate得到修改后的，继续挑选想要的使用笔刷涂改想要修改的部分，再点击Generate得到修改后的，重复该过程直到得到自己满意的图像为止。

  2. 修改眼袋太大的问题，点击Send to extras ，将GFPFAN visibility和Code Former visibility调整为0.6和0.75，再点击Generate得到修改后的。后面可以继续在Send to inpaint中使用笔刷涂改想要修改的部分。

  1. 修改完图片后，点击Send to extras放大图片，将GFPFAN visibility和Code Former visibility都调整为0.1，再点击Generate得到高解析度的AI网红照片。

AI网红照片展示，设备不行，懒得调的了。

AI Coser照片的生成

  • 与AI网红照片的生成一样，点击 Lucy画像模型， 添加 Lucy 的触发词lucy \(cyberpunk\)，在Lucy (Cyberpunk Edgerunners) LoRA - 35 epochs找到网友的给出的 Lucy 照片和对应的提示词，将网页分享的提示词也加入提示词栏；

  • 在Chillout Mix-NI模型下方的网友贴图中选择脸部的提示词，将其复制到提示词栏中。

  • 再generate抽奖，还可以将 korean doll likeness、taiwan doll likeness、japanese doll likeness画像模型也加入提示词栏中。注意调整权重，Lucy 的权重大些。

  • 继续generate抽奖。

  • 现在脸型调整好后，需要调整 Lucy 的服装，删除提示栏中的提示词，保留触发词和权重。

  • 在Lucy (Cyberpunk Edgerunners) LoRA - 35 epochs找到网友的给出的 Lucy 全身照和对应的提示词，将相关 Lucy 服装的提示词加入提示词栏，继续加大 Lucy的权重，继续generate抽奖。

  • 在img2img中 需要持续调整Denoising strength来改变生成图像，最后Denoising strength设置为0.5 来生成最后的图像。
